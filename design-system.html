Overview.

Establishing a design system brings significant benefits for a product team. It promotes consistency, clarity, and quality in the design process. It can also deliver significant competitive advantages for businesses.

MP design system is a on-going project. Typgraphy, brand colors, logo are defined. I have used these things here, goldern ratio, color harmony, color wheel..etc

My Role

UX Designer

Project Status

On-going

Project Duration

Four Weeks

Typography.

Typography is the medium of communication. The basic approach of using typography is selecting the correct font. The font should be as clean as possible. It shouldn't be too small and crummy. Here I have used the goldern ratio (1.61) to build the scale.

Colors.

Color can set the basic tone, mood, connotation and conception of a brand or a product. Here I have used canva color wheel for creating this color pallete. 60-30-10 color rule is used to set the theme. Figma color contrast plugin is used to check the accesbility for those colors. Assebility is passed.
